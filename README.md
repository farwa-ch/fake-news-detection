# Fake News Detection System (AI & ML Project)

## Introduction
The project uses machine learning and natural language processing to detect whether the news content is real or fake.

## Technologies Used
- Python
- Scikit-learn
- Natural Language Processing (TF-IDF)
- Streamlit

## Key Features
- Detect fake news articles based on text
- Machine Learning classifier algorithm
- Web application interface

## Instructions for Execution
pip install -r requirements.txt
python src/train_model.py
streamlit run app/app.py

## Dataset
Kaggle Fake News Dataset
